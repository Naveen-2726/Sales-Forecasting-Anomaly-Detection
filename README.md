# Sales Forecasting & Anomaly Detection (AI Project)

## Tools Used
- Power BI Desktop
- DAX
- AI Visuals (Forecast, Anomalies, Smart Narrative)

## Project Objective
To move from reactive reporting to predictive analytics by forecasting future sales, detecting unusual patterns, and generating AI-driven insights.

## Key Features
- Historical sales trend analysis
- Seasonality identification
- AI-based sales forecasting
- Anomaly detection with explanations
- Smart Narrative for automated insights

## Dataset
Retail sales dataset (Order Date, Sales, Category, Region).

## Insights
- Sales show seasonal behavior with strong year-end growth.
- AI forecast predicts continued growth.
- Anomalies highlight unusual sales spikes/drops needing investigation.
